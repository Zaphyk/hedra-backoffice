Discover and explore colorful new worlds in an open-world setting with procedurally generated quests


Project Hedra is a roguelike game with RPG elements: open-world, replayable and quest-oriented are some of the features which best describe Project Hedra. The game is divided into "runs" where each world is procedurally generated and can vary in theme, appearance, natural complexity levels ( such as the variety of flora in the world and the weather ) along with many other factors which make each run a unique challenge.
